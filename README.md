## 你知道吗 (Do You Know?)

一个简单的小网页。展示一些奇奇怪怪的冷知识。

### 介绍

随机显示知识文本, 点击刷新按钮(🔄)刷新。

所有文本位于`facts.js`中。每次刷新会确保显示不同的文本。

点击右侧About中的链接或[点击这里](https://pengyuhan17.github.io/do-you-know/)进入GitHub Pages网页。
